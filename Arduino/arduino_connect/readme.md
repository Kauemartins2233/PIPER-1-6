# Arduino Connect

Firmware para **Arduino + Ethernet Shield** que implementa os controladores PI discretos da aeronave Piper 1/6 com comunicacao UDP para o X-Plane.

## Hardware

- Arduino (Mega/Uno) com Ethernet Shield (W5100/W5500)
- Comunicacao via SPI/Ethernet UDP

## Arquivo

| Arquivo | Descricao |
|---------|-----------|
| `arduino_connect.ino` | Sketch principal com controladores PI, recepcao de telemetria e envio de comandos |
