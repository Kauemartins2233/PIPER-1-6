# Arduino Connect

Firmware para **Arduino + Ethernet Shield** que implementa os controladores PI discretos da aeronave Piper 1/6 com comunicação UDP para o X-Plane.

## Hardware

- Arduino (Mega/Uno) com Ethernet Shield (W5100/W5500)
- Comunicação via SPI/Ethernet UDP

## Arquivo

| Arquivo | Descrição |
|---------|-----------|
| `arduino_connect.ino` | Sketch principal com controladores PI, recepção de telemetria e envio de comandos |
