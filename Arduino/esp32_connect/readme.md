# ESP32 Connect

Firmware para **ESP32** com Wi-Fi que implementa os controladores PI discretos da aeronave Piper 1/6 com comunicação UDP para o X-Plane.

## Hardware

- ESP32 (qualquer variante com Wi-Fi)
- Comunicação via Wi-Fi UDP

## Arquivo

| Arquivo | Descrição |
|---------|-----------|
| `esp32_connect.ino` | Sketch principal com controladores PI, recepção de telemetria e envio de comandos |
