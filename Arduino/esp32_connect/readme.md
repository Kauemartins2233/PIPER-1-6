# ESP32 Connect

Firmware para **ESP32** com Wi-Fi que implementa os controladores PI discretos da aeronave Piper 1/6 com comunicacao UDP para o X-Plane.

## Hardware

- ESP32 (qualquer variante com Wi-Fi)
- Comunicacao via Wi-Fi UDP

## Arquivo

| Arquivo | Descricao |
|---------|-----------|
| `esp32_connect.ino` | Sketch principal com controladores PI, recepcao de telemetria e envio de comandos |
